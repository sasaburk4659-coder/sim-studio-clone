# 🚀 Sim Studio Clone — AI Workflow Builder
      
      Полнофункциональный клон Sim Studio — визуальный конструктор AI-автоматизаций в одном HTML файле.
      
      ## ✨ Возможности
      
      - 🎨 Визуальный конструктор — drag & drop блоки на canvas
      - 60+ типов блоков — Agent, Function, API, Gmail, Slack, GitHub и др.
      - 🤖 AI Chat — встроенный ассистент (подключите Google Gemini API)
      - ⚡ Управление workflows — создание, дублирование, экспорт JSON
      - 🔗 Соединения между блоками с анимацией
      - 💾 Auto-save в localStorage
      - ↩️ Undo/Redo (Ctrl+Z / Ctrl+Y)
      - 🔍 Zoom & Pan
      - 📋 Copy/Paste блоков
      - 🖱️ Context Menu
      - 📊 Execution Logs
      - 🗺️ Minimap
      - ⚙️ Settings с API ключами
      
      ## 🚀 Запуск
      
      Просто откройте index.html в браузере. Никаких зависимостей!
